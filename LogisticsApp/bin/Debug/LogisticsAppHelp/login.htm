Login Form

In this form the user is able to connect to the application

What happens if I don't already have an account

"press here" -> insert username -> insert password -> retype password -> "Create account"

How to make a successfull login

Make sure that you type your username and password correctly in the respective fields.

Buttons and their uses

button "Exit": Press to exit the appliction

button "Connection": Press to connect

button "press here:" Press to open the form for account creation

button "Create Account:" Press to create account

button "Cancel:" Press to cancel
